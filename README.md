# Sri Lanka Airport Website (BIA)

A modern React + Tailwind website UI for **Bandaranaike International Airport (BIA)**.

## Tech
- React (Create React App)
- Tailwind CSS
- react-router-dom
- lucide-react icons

## Getting started

### Install
```bash
npm install
```

### Run (dev)
```bash
npm start
```

### Build (production)
```bash
npm run build
```

## Notes
- Static output builds to `build/`.
- `node_modules/` and `build/` are ignored via `.gitignore`.
